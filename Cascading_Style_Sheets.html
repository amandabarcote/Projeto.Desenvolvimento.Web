Cascading Style Sheets - CSS


O que é o CSS?

A sintaxe é constituída pelo seletor, que é a tag do HTML a ser modificada. A tag escolhida terá as modificações programadas no CSS. O seletor possui as chaves, que delimitam o início e o fim do código. A propriedade a ser alterada deve ser explicitada entre as chaves, com dois pontos “:” no final para definir o valor. Após definir o valor, ele deve ser fechado com ponto e vírgula “;”. Os comentários são definidos entre os caracteres “/* */”. Quando se tem uma subpropriedade, ela deve ser colocada logo após a propriedade, separada por um traço “-”.


Formas do CSS

O CSS pode ser utilizado de três formas: “Inline”, que é definido dentro da linha da tag; “Incorporado”, que é definido dentro da tag head; e o “Externo”, que é um documento à parte. No caso do “Externo”, você escreve o CSS uma única vez e pode usá-lo em qualquer página, fazendo um link com o documento externo. As prioridades que o navegador atribui a cada estilo seguem a ordem mencionada: Inline, Incorporado e Externo.
